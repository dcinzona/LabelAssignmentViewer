Salesforce Label Assignment Viewer
Component Overview

The Label Assignment Viewer is a Salesforce Lightning Web Component (LWC) that displays UserDefinedLabelAssignment records related to a selected UserDefinedLabel in a standard Salesforce related list format.

Features
✓
Label selector using a combobox with counts of related assignments
✓
Datatable with sortable columns and special formatting for Case records
✓
SubjectOrName field as first column with clickable links and object-specific icons
✓
Row-level actions to delete assignment records or view target records
✓
Search functionality to filter assignments
✓
Responsive design with proper empty and loading states
✓
Select multiple records and delete in bulk with confirmation dialog
✓
Delete entire labels (including all assignments) with confirmation dialog

Implementation Details

The component consists of three key files:

  • labelAssignmentViewer.js - The JavaScript controller handling data fetching, processing and user interactions
  • labelAssignmentViewer.html - The HTML template defining the component's structure and appearance
  • LabelAssignmentController.cls - The Apex controller providing data services

Special handling is implemented for Case records, where the SubjectOrName field is dynamically formatted to show the CaseNumber and Subject. Object-specific icons are shown based on the record type, and users can interact with records via row actions.
